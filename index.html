Selectoption

Selectoption is small jquery plugin for selecting single/multiple options of select element. Selecting option is a very common use case and below are some of them

  1. Developer A need to select single option in some pages
  2. Developer A need to select multiple option in some pages where selections are in array.
  3. Developer A need to select multiple option in some pages where selections are in Json format.

This plugin cover all of them.

How it works ?

Developer A need to select single option in some pages.
$("select").Selectoptions({ data:2})

Developer A need to select multiple option in some pages where selections are in array. $("select").Selectoptions({ data: [1, 2] });

Developer A need to select multiple option in some pages where selections are in Json format.
$("select.selectmultiplejson").Selectoptions({ comparewith: "label", datasource: "json", data: [{ label: "1", Caption: "One" }, { label: "3", Caption: "Three"}] })

Parameters

datasource : singlevalue,array,json.
data : data to be selected.
comparewith : Label of json object used to compare with option value.
